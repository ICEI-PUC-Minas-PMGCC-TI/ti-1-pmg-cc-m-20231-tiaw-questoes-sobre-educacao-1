1) Exercícios sobre polinômios

Questão 1

A divisão do polinômio P(x) pelo polinômio D(x) = x² + 2x – 5 gera como quociente o polinômio Q(x) = 3x – 1 e o resto R(x) = 5x + 2. Então, o polinômio P(x) é:


A) P(x) = 3x³ + 5x² - 12x + 5

B) P(x) = x³ + 5x² - 17x + 5

C) P(x) = 3x³ + 6x² - 15x + 7

D) P(x) = 2x³ - 4x² + 12x + 7

E) P(x) = 3x³ + 5x² - 12x + 7

Avalie a questão
0 0 0 0 0

Resposta da questão 1

Alternativa E.


Sabemos que:

P(x) = D(x) ⋅ Q(x) + R(x)

Logo, temos que:

P(x) = (x² + 2x – 5)(3x – 1) + (5x + 2)

P(x) = (3x³ + 6x² - 15x – x² – 2x + 5) + (5x + 2)

P(x) = (3x³ + 5x² - 17x + 5) + (5x + 2)

P(x) = 3x³ + 5x² - 12x + 7
